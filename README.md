## کیبورد مجازی

# توضیح کلی
این پروژه یک صفحه کلید مجازی است که با جاوااسکریپت ساخته شده.
کاربر می تواند بدون استفاده از صفحه کلید فیزیکی، با کلیک روی دکمه های صفحه کلید مجازی، متن خود را وارد کند.

# امکانات پروژه
· نمایش کاراکترها با کلیک روی هر کلید
· دکمه Backspace:
   · کلیک معمولی: یک کاراکتر از متن کسر می کند
   · کلیک راست: کل متن را پاک می کند
· دکمه Caps Lock: متن همه کلیدها را بین حروف بزرگ و کوچک تغییر می دهد
· دکمه Enter: (در حال حاضر عملکرد خاصی ندارد، ولی جای توسعه دارد)
